## ✨ Overview

**SnippetFlow** is a lightning-fast web application designed for developers who hate rewriting boilerplate. Whether it's a complex CSS animation, a robust React Hook, or a Python utility script, SnippetFlow provides instant access to tested, high-quality code fragments.

### Why SnippetFlow?

- **Instant Search:** Find exactly what you need with our fuzzy-search algorithm.
- **One-Click Copy:** Zero-friction workflow—click once, paste into your IDE.
- **Categorized:** Browsable sections for React, CSS, JavaScript, SQL, and more.
- **Developer Centric:** Built by developers, for developers, with a focus on speed and clarity.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm / yarn / pnpm

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/yourusername/snippetflow.git
   cd snippetflow
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
